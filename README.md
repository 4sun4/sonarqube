```bash
npx react-native run-ios
npx react-native run-android

```

## Run the project from Xcode

- open Xcode 
- open the project file `RecruitOnline.xcworkspace` from `ios/` folder
- choose any simular / device if available
- run the project

## Run the project from Android Studio

- open Android Studio 
- open the android project folder `android` in Android studio
- choose any emulator / device if available
- run the project
